## 1. 简介

减少自组自行车的试错成本，并介绍一些好用的配件，以及一些骑车技巧。

## 2. 通勤公路车

自组通勤公路自行车，兼顾休闲骑行和业余比赛。

三款都是减震座管，可以做到不穿骑行裤也能长距离骑行。

三款用的都是圈刹，相比碟刹有好维护、重量轻、便宜、制动力强等特点，并且都配置了全天候刹车皮，以减少雨天刹车性能问题，不走泥路越野，性能完全没有问题。
